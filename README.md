# :watch: effort

:rocket: blazing fast :rocket: tool to log work hours based on time frames written in rust

## :musical_keyboard: Keybinds

| key     | action               |
| ------- | -------------------- |
| `j`/`k` | navigate up and down |
| `o`     | insert new activity  |
| `e`     | edit activity        |
| `s`     | toggle statistics    |
| `f`     | add day off          |
